Information Disclosure Vulnerability Report
Google Dorking-based Information Disclosure Vulnerability Report
Basic Information

Target Domain: example.com

Discovery Date: 2025-07-15 14:30:00

Severity: Medium

Vulnerability Type: Information Disclosure

Google Dork Used
Search Query
site:example.com (filetype:env | filetype:config | filetype:sql | filetype:log | filetype:json | filetype:yml | filetype:csv) ("password" | "secret" | "api_key" | "database" | "backup")
Vulnerability Details
Exposed URL
https://example.com/backup/config.env
Exposed Information
Sensitive configuration files including database connection information, API keys, and administrator account details are exposed and accessible from external sources
Impact and Risk Assessment

• Potential unauthorized database access

• API key theft leading to service abuse

• Internal system information exposure

• Foundation for additional attacks

Recommended Remediation

1. Immediately delete or restrict access to exposed files

2. Change all exposed account credentials and API keys immediately

3. Disable web server directory listing functionality

4. Block access to sensitive files through .htaccess or server configuration

Verification Method

1. Enter the Google Dork query in Google search

2. Check exposed files for the target domain in search results

3. Verify file contents by directly accessing the URL
